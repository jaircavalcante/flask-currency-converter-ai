## Prompt utilizado para geração do projeto.

Atue como um desenvolvedor sênior Python.

Crie um projeto web completo utilizando Flask que funcione como um conversor de Real (BRL) para Dólar (USD).

Especificações:

Arquitetura:
- Estrutura organizada em pastas (app/, templates/, static/)
- Separação de responsabilidades
- Endpoint RESTful

Backend:
- Flask
- Consumo de API pública e gratuita para cotação do dólar em tempo real
- Tratamento de exceções
- Retorno em JSON
- Logs básicos

Frontend:
- Interface moderna com CSS responsivo
- JavaScript usando Fetch API
- Atualização dinâmica sem reload da página
- Loader enquanto aguarda resposta
- Validação de input

Extras:
- README.md com instruções
- requirements.txt
- Comentários explicativos no código
- Código pronto para rodar

## Rodar o projeto
1. python -m venv venv
2. source venv/bin/activate  
3. pip install -r requirements.txt
4. python3 ou python app.py
5. Executando em: http://127.0.0.1:5000
