# Currency Converter AI - Specification

## Objetivo
Criar uma aplicação web em Flask que converta Real (BRL) para Dólar (USD)
utilizando cotação em tempo real via API pública gratuita.

## Funcionalidades
- Campo para inserir valor em BRL
- Botão de conversão
- Retorno do valor convertido
- Interface moderna
- Atualização dinâmica sem reload

## Requisitos Técnicos
- Flask
- Consumo de API pública
- Endpoint /convert retornando JSON
- Organização MVC simplificada
- Tratamento de erros
